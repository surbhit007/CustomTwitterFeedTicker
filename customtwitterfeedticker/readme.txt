== Description ==

Custom Twitter Feed Ticker, list the twitter feed and run feed in ticker slider.

This plugin is very helpful for plugin and theme developers.

* It adds a settings page to "Dashboard"->"CUSTOM Twitter Feed Settings" where you can simply add your twitter api configuration settings.
* Custom Twitter Feed Ticker fetcu feed from twitter using @username and display them as ticker slider.

Usage :

1. Download and extract this plugin to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. "Dashboard"->"CUSTOM Twitter Feed Settings"
